EnergyDB
========

EnergyDB is a facade over `TimeDB <https://github.com/rebase-energy/timedb>`_
that adds arbitrary-depth asset hierarchies (Portfolio → Site → WindFarm →
WindTurbine → …) with time-series routing. Users interact with EnergyDB only;
TimeDB is managed internally through composition.
